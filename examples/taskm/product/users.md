# User Personas

## Primary Persona — The Solo Developer

**Name:** Alex
**Role:** Software developer or technical knowledge worker
**Setup:** Works from a personal laptop, comfortable running a local web app

### Goals
- Maintain a single place for all personal tasks without signing up for yet another SaaS tool
- Quickly dump tasks out of their head and come back to them
- Mark things done and keep a clean list

### Frustrations
- Apps that require an account just to store a to-do list
- Cluttered UIs with features they'll never use
- Slow or unreliable sync when offline

### Behaviors
- Opens the app in a pinned browser tab
- Adds tasks throughout the day, reviews and ticks them off at end of day
- Prefers keyboard shortcuts over mouse clicks
